CRUD and Web Scraping URL

Through a local database and a tool for testing RESTful services through HTTP request submissions you will be able to:

  • Create a user that contains:
    • "username" = username,
    • "first_name" = first_name,
    • "last_name" = last_name,
    • "email" = email

    • A user using the id generated at creation:
    • Example: "localhost:5000/"id""

  • Update a user

  • Delete a user using their id
    • Example: "localhost:5000/users/"id""

  • you will also be able to typing in the address bar a website starting with:
    https://afterlocalhost:5000/and having an array return with the list of urls that particular website has in its TAG 'a' and extracting the inside 'href' content. Example: "localhost:5000/https://wwww.site.com"
